1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework ?

    2. Adanya framework dalam bahasa pemrograman dapat membantu programmer untuk membuat aplikasi lebih cepat dan efisien karena di dalam framework terdapat banyak fitur dan function yang dapat mempermudah programmer.

  2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?

    3. If Else, statement yang digunakan untuk mengatur alur program berdasarkan kondisi tertentu.
    While, statement yang digunakan untuk melakukan perulangan selama perintah kondisi terpenuhi.
    Do While, statement yang mirip dengan while hanya saja dalam blok do akan dijalankan minimal satu kali sebelum mengecek kondisi while.
    Switch Case, statement yang digunakan untuk memeriksa suatu nilai dan mengambil keputusan berdasarkan nilai tersebut.
    For, statement yang digunakan untuk perulangan dengan jumlah perulangan sebanyak yang ditentukan.
    Foreach, statement yang digunakan untuk melakukan perulangan berupa array.

  3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?

    4. Constructor, sebuah method khusus yang akan dipanggil ketika Objek telah dibuat.
    Method, sebuah function yang terdapat di dalam class.
    Class, sebuah blueprint untuk membuat objek.
    Object, sebuah instance dari class.
    Parameter, variabel yang digunakan untuk menerima input dari luar function.
    Function, blok kode yang digunakan untuk melakukan tugas tertentu dan dapat mengembalikan nilai.
    Procedure, blok kode yang digunakan untuk melakukan tugas tertentu namun tidak dapat mengembalikan nilai.

  4. Perbedaan method="Post" dan "Get" ?

    5. Method GET mengirimkan data melalui url, sehingga data dapat terlihat di url browser. Sedangkan method POST mengirimkan data tidak melalui url melainkan melalui http request.

  5. Jelaskan konsep mengenai web service dan fungsi apa yang digunakan untuk memanggil web service client ?

    6. Konsep web service yaitu mengizinkan sistem client untuk mengakses layanan tertentu tanpa harus memahami bagaimana layanan tersebut bekerja. Client dapat menggunakan curl untuk mengirim request dan mengambil data.

  6. Jelaskan mengenai MVC dan HMVC !

    7. MVC (Model-View-Controller) adalah sebuah konsep arsitektur yang terdiri dari tiga bagian yaitu, model, view dan controller. Sedangkan HMVC merupakan arsitektur tingkat lanjut dari MVC. HMVC dapat memisahkan logika bisnis dan fungsi tertentu ke dalam modul yang terpisah.

  7. Sebutkan fungi curl() dalam php!

    8. curl adalah sebuah function yang digunakan untuk membuat request.

  8. Bagaimana menampilkan sebuah element array dalam PHP ?

    9. Dengan mengakses index array. Index array dapat diakses dengan menggunakan tanda [index]

  9. HTTP dan HTTPS ?

    10. HTTP dan HTTPS adalah sebuah protokol yang digunakan untuk mengirim sebuah request dari client ke server. Perbedaan dari HTTP dan HTTPS adalah tingkat keamanannya, HTTPS lebih aman dari HTTP.

  10. Jelaskan Apa itu API? sebutkan contohnya?

    11. API (Application Programming Interface) adalah cara antara sistem yang berbeda untuk berkomunikasi satu sama lain. Salah satu contoh API adalah REST, SOAP.
